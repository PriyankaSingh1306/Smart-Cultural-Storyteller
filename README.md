# Smart-Cultural-Storyteller
1. Developed an AI storytelling system to generate multilingual cultural stories using Google Gemini API.
2. Integrated Stable Diffusion and text-to-speech APIs to create story visuals and audio narration.
3. Built an interactive Streamlit interface with comic-style storytelling experience.
4. Improved usability through modular design, session state management, and UI enhancements.
